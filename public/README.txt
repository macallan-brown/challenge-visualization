Challenger Visualization
Created by Macallan Brown

To view my visualization on a Mac Computer with Python3 installed:
1) Open terminal
2) Navigate to ‘challenge-visualization’ directory
	cd challenge-visualization
3) Start a Python3 Server
	python3 -m http.server
4) Go to ‘http://localhost:8000/‘ in your favorite web browser (Chrome?).
5) Enjoy!

P.S Click the blue button to switch between the two types of visualizations.
