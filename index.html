Challenger Visualization

This visualization uses data coming from the 23 shuttle flights prior to the challenger disaster of 1986. The variables four attributes visualized for each launch are:

  • Flight Index: The temporal ordering of the flights.
  • Number of O-Rings Distressed
  • Launch Temperature (Degrees Fahrenheit)
  • Leak Check Pressure (PSI)

For more infromation on the dataset please visit UCI Machine Learning Reposity .
